# Safety

## Current safety system

- User reports
- User blocks
- Moderator notes and assignments
- Enforcement actions and enforcement reviews
- Member safety-state summaries
- Queue pressure and SLA tracking
- Web moderation dashboard with member detail views
- Admin health monitoring for the moderation data path

## Protection

- Profile and onboarding gates before queue access
- Blocked-user exclusion in matchmaking
- Queue and moderation previews for better operator context
- Same-origin admin proxy routes for moderation reads
- Exportable moderation snapshots and summaries

## Operational workflow

- Moderators review open cases
- Leads and admins can escalate, reassign, and export
- SLA alerts call out overdue first-action and resolution work
- Admin health warnings gate risky bulk actions when the data path is degraded

## Future

- AI toxicity detection
- Scam detection
- Automated notifications or alerts for high-risk moderation conditions
